Restauration de l'ancien comportement (avant la 2.5.6.53756) de la fonction copier/coller de l'Editeur de Mission.
Si le nom d'une unité se termine en #n avec n un numérique, alors l'Editeur de Mission incrémentera simplement la partie numérique en collant la nouvelle unité.
Sinon, le nouveau mécanisme sera utilisé (remplacement du nom de l'unité par un nouveau nom basé sur le nom du groupe, en ignorant complètement le nom de l'unité)

→ https://forums.eagle.ru/showthread.php?p=4461982

Restores the old (ante 2.5.6.53756) behavior of the Mission Editor's copy/paste of groups and units.
If a unit name ends with #n where n is an numeral, then the ME will simply increment the numeric part when pasting.
Else, the new mechanism is used (replacing the unit name with a new name based on the group name, completely ignoring the unit name)
