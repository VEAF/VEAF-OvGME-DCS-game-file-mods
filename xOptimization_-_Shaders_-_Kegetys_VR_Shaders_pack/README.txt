A shader pack that gives a better performance, specially in VR.
Read the original_readme.txt file.
TL;DR:
- Remove terrain metacaches in the game install directory.
  In each terrain folder, delete the "misc\metacache\dcs" folder. For example: "Mods\terrains\Caucasus\misc\metacache\dcs"
- When you run DCS World, the shaders will be compiled as they are used for the first time.
  First startup and loading of mission will take a while (can be 10 minutes or more). 
  While playing you can also see some stutters when new shaders are compiling for the first time.

Un pack de shaders qui améliore la performance, particulièrement en VR.
Lisez le fichier original_readme.txt.
- Effacez les metacaches des terrains dans le répertoire d'installation du jeu.
  Dans chaque répertoire de terrain, effacer le répertoire "misc\metacache\dcs"; par exemple, "Mods\terrains\Caucasus\misc\metacache\dcs"
- Le premier démarrage de DCS sera très long (jusqu'à 10 minutes). C'est normal !
